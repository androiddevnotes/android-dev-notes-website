# Using testTag in Maestro as Resource IDs to identify a composable

## Overview

To identify a composable:

```kotlin
Button(
        text = "hello",
        modifier = Modifier.semantics { testTagsAsResourceId = true }.testTag("test_tag_button_hello")
    )
```

In Maestro Studio, you can then identify above composable via testTag `test_tag_button_hello`.

Notice, `testTagsAsResourceId = true` is required to make it work.
