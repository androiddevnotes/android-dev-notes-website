# BuildKonfig - BuildConfig for Kotlin Multiplatform Project

## Overview

BuildKonfig is a Kotlin Multiplatform Project tool that generates BuildConfig for easy sharing of values between platforms. It is a Gradle plugin.

With support for embedding values from Gradle files and targeting specific platforms, BuildKonfig is a powerful tool for simplifying multiplatform development.
