# Relay - Figma to Jetpack Compose

## Overview

The Relay plugin for Figma helps designers and developers create and maintain design systems for Jetpack Compose Android apps by allowing them to create and organize UI components in a UI Package, which can be exported and converted to Jetpack Compose code automatically using Relay in Android Studio.
