# ImageResizer - A Jetpack Compose Image Resizer and Color Picker Android App

## Overview

The app is designed to resize image & pick colors.

## Tech Stack

- Jetpack Compose
- MVVM
- Compose Navigation Reimagined
- Coil
- Palette
- Compose Extended Gestures
- Compose Screenshot
- Compose Extended Colors
- Compose Image
- Material
- Flow

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

 ```
 git clone https://github.com/T8RIN/ImageResizer
    ```
2. Open the project in Android Studio.

3. Build and run the project.

## Summary

The app has lots of features like quality, resize, color picker, image type conversion, rotation etc. The app has simple UI and UX and is easy to use.
