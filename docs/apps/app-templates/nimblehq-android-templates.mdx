# nimblehq Android Templates - Optimized and modern in XML and Jetpack Compose

## Overview

A collection of well-made & modern Android templates by [nimblehq](https://github.com/nimblehq) for Jetpack Compose and XML.

These templates serve as skeleton to start a new project generated via kscript.

## Tech Stack

- Jetpack Compose
- XML
- kscript
- Hilt
- MVVM
- Chucker
- Coroutines
- Navigation
- Retrofit
- Mockk
- Kotest
- Detekt
- Moshi

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

    ```
    git clone https://github.com/nimblehq/android-templates
    ```

2. cd into the scripts folder and run the following command for XML template:

    ```
    kscript new_project.kts package-name=co.myxmlproject.example app-name="My XML Project" template=xml
    ```

    or for Compose template:

    ```
    kscript new_project.kts package-name=co.mycomposeproject.example app-name="My Compose Project" template=compose
    ```

3. Open the project in Android Studio and run it.

4. Also, optionally check the samples directory for more idea on how to use the template.
