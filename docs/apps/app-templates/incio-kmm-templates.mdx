# Incio - Template project Kotlin Multiplatform for Cross-Platform Mobile Development

## Overview

Modern templates Kotlin Multiplatform for Cross-Platform Mobile Development by
[NBSMobile](https://github.com/nbsmobile). This template following the best
development practices.

### Module

- `shared` data and domain layer
- `iosApp` ios presentation layer
- `androidApp` android presentation layer

## Build Instructions

To get started with this project, follow these steps:

1. Follow the
   [KMM Guide by jetbrains](https://kotlinlang.org/docs/multiplatform-mobile-getting-started.html).

2. Install Kotlin Multiplatfrom Mobile plugin in Android Studio.

3. Clone the repository to your local machine using the following command:

```
git clone https://github.com/nbsmobile/incio.git
```

4. Rebuild Project

5. To run in iOS, `Open Xcode`, select `.xcworkspace`, then `pod install` inside
   `iosApp` folder to install shared module and ios dependencies
