# Focus Launcher - A minimal Android Launcher app in Jetpack Compose

## Overview

A minimal Android Launcher app in Jetpack Compose.

Focus Launcher is a sleek and minimalist Android launcher app that provides a clutter-free interface for a distraction-free experience. The launcher offers various features such as Dark mode, Lunar Phase info, Fancy animated Clock, Favorite apps, and Hide apps to customize home screen.

Apps are searchable!

## Tech Stack

- Compose
- MVVM
- Hilt
- Unit Testing
- Room
- DataStore
- Navigation
- Ktor
- Turbine
- Material
- Flow

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

    ```
    git clone https://github.com/mslalith/focus_launcher
    ```

2. Open the project in Android Studio.

3. Build and run the project.

That's it! You're ready to go.
