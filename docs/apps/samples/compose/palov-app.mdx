# Palov - A Recipe App in Jetpack Compose

## Overview

The app is designed to provide food, ingredients and recipe information sourced
from Spoonacular API

## Tech Stack

- Compose
- MVVM
- Hilt,
- Unit Testing
- Retrofit
- Room
- Navigation
- jsoup
- Flow

## Code

You can find the complete source code for this Palov App on
[GitHub](https://github.com/samandar-me/palov). The project is open source.

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

   ```
   git clone https://github.com/samandar-me/palov
   ```

2. Open the project in Android Studio.

3. Build and run the project.

That's it! You have a information food, ingredients and recipe app built using
Jetpack Compose.
