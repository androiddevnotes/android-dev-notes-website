# KShell-UI - A Shell App in Jetpack Compose

## Overview

The app is an interface to interact with the device's file system and executes
various commands.

## Tech Stack

- Compose
- Coroutine
- Flow
- Room
- Material

## Code

You can find the complete source code for this KShell UI on
[GitHub](https://github.com/saidooubella/KShell-UI). The project is open source.

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

   ```
   git clone https://github.com/saidooubella/KShell-UI
   ```

2. Open the project in Android Studio.

3. Build and run the project.

That's it! You can write command in app with beautiful UI.
