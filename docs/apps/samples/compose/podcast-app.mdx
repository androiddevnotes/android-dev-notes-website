# Podcast App - Made with Jetpack Compose and ExoPlayer

## Overview

Android Podcast app made with Jetpack Compose and ExoPlayer. Podcast infromation
provided by [Listen Notes API](https://www.listennotes.com/).

## Tech Stack

- Jetpack Compose
- ExoPlayer
- Glide
- Hilt
- Retrofit
- Navigation
- ViewModel
- DataStore
- Palette API

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

```
git clone https://github.com/fabirt/podcast-app
```

2. Open the project in Android Studio

3. Build and run the project.

That's it! You can learn about jetpack compose and exoplayer.
