# Apps - Introduction

Apps section is place for fascinating open source apps that you have found useful and think others will find useful too. You are encouraged to add your own interesting apps to the Apps section!

## How can I add content to the Apps section?

In the Apps section, you can add open source apps that you have found useful and think others will find useful too.

Example: Found an okay app with modern architecture and want to share it with others? Add it to the Apps section.

Check out the [GitHub repository](https://github.com/androiddevnotes/android-dev-notes-website) page for more information.
