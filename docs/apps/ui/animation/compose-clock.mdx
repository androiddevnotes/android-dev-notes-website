# Build an Analog Compose Clock with Particle Effect in Jetpack Compose

## Overview

If you're looking for a fun project to build in Jetpack Compose, consider building an analog clock with a particle effect! This clock features a clock face built using the Canvas API, and a particle effect inspired by project of the Flutter Clock challenge.

## Tech Stack

- Compose

## Code

You can find the complete source code for this clock on [GitHub](https://github.com/manuel-martos/compose-clock). The project is open source.

## Build Instructions

To get started with this project, follow these steps:

1. Clone the repository to your local machine using the following command:

    ```
    git clone https://github.com/manuel-martos/compose-clock
    ```

2. Open the project in Android Studio.

3. Build and run the project.

That's it! You should now have a working analog clock with a particle effect built using Jetpack Compose.
