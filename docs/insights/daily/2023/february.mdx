# Daily Insights - February 2023

Found something interesting? Discover something new? Share it with the community!

## February 15, 2023

[Relay - Figma to Jetpack Compose](https://developer.android.com/jetpack/compose/tooling/relay/basic-tutorial)

With Relay by the Android Team, you can design UI components in Figma, import them in Android Studio and Jetpack Compose code is auto-generated!

[Kotlin Onboarding: Introduction](https://plugins.jetbrains.com/plugin/21067-kotlin-onboarding-introduction)

A new Kotlin course for beginners by JetBrains. The course is in the form of a project that's divided into small tasks. The course explains basic Kotlin concepts. The course is integrated into the Intellij Idea IDE (Free)

[BroadcastChannel deprecation in Kotlin Coroutines](https://github.com/Kotlin/kotlinx.coroutines/issues/3621#issuecomment-1427782338)

In the issue that discusses Fast and scalable channels algorithm, the maintainer commented:

> This issue is also a final straw for the *BroadcastChannel deprecation -- broadcasts were never meant to be complete channels (-> comply its interface) and it's time for us to let them go and stop maintaining them"
