## February 23, 2023

* [BoringLayout](https://developer.android.com/reference/android/text/BoringLayout)

A BoringLayout is a type of Layout that only supports left-to-right text that fits on a single line. It's a very basic implementation and typically not something you would create yourself. However, if you do need to create one, it's important to check if the text meets the requirements by calling the isBoring(CharSequence, TextPaint) method first.

## February 17, 2023

[ViewModel APIs cheat sheet](https://developer.android.com/topic/libraries/architecture/viewmodel/viewmodel-cheatsheet)

The cheat sheet shows ViewModel APIs in Jetpack with artifact location, the scope of returned instance, and example usage presented for quick reference. Includes Jetpack Compose, Core ViewModel API, Activity, Fragment, etc

## February 15, 2023

[Relay - Figma to Jetpack Compose](https://developer.android.com/jetpack/compose/tooling/relay/basic-tutorial)

With Relay by the Android Team, you can design UI components in Figma, import them in Android Studio and Jetpack Compose code is auto-generated!
